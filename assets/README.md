# 图片素材约定

页面已经带了一套占位图。你后续直接覆盖同名文件即可，不需要改代码。

```text
assets/avatar.jpg
assets/hero.jpg
assets/character.png
assets/covers/anime-blog-redesign.jpg
assets/covers/frontend-ai-tooling-notes.jpg
assets/covers/static-blog-care.jpg
assets/covers/night-debugging-room.jpg
assets/covers/interface-mood-board.jpg
```

建议尺寸：

- `avatar.jpg`：正方形，至少 512 x 512。
- `hero.jpg`：横向大图，建议 1600 x 1000 或更高。
- `character.png`：透明背景立绘，建议高度 1200px 以上。
- `assets/covers/*.jpg`：文章封面，建议 1200 x 800。
